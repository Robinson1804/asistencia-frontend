# **App Name**: AsistenciaYA

## Core Features:

- Employee List Display: Fetch and display a list of employees from the 'CONTROL DE ACCESO' project's database.
- Attendance Marking: Allow marking individual employees as 'Presente', 'Tardanza', or 'Falta'.
- Attendance Logging: Record the attendance status (Presente, Tardanza, Falta) and timestamp in the database for each employee.
- Attendance Summaries: Generate summaries (e.g. totals for registered, present, tardy, or absent).

## Style Guidelines:

- Primary color: Dark blue (#293B5F) to give a professional feel.
- Background color: Light gray (#D3D3D3) for a neutral backdrop.
- Accent color: Teal (#2E8A99) to add a touch of vibrancy and signal interactivity.
- Body and headline font: 'Inter', sans-serif, for a modern, machined, objective, neutral look.
- Use clear and professional icons for attendance status.
- Display the list of employees in a card-based layout, each with a clearly visible attendance status selector.
